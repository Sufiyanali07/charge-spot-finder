EV Charging Station Management

Backend API is running at http://localhost:5001

The full Vue.js frontend is still being set up. In the meantime, you can test the API endpoints:

  • GET /api/stations - List all charging stations
  • POST /api/auth/register - Register a new user
  • POST /api/auth/login - Login and get JWT token

You can use tools like Postman or curl to test these endpoints.
